# 📘 Exercício 02: Controle de Fluxo e Classificação de Pedidos

## Objetivo

Simular a etapa de **transformação baseada em regras de negócio** em um pipeline de dados. O script percorre uma coleção de registros, aplica classificações condicionais, acumula métricas e gera um relatório consolidado, mantendo o código limpo, legível e escalável.

## Conceitos Aplicados

- Estruturas condicionais (`if/elif/else`) para validação de regras
- Laços de repetição (`for`) sobre lista de dicionários
- Acumuladores e contadores (padrão de agregação manual)
- Convenção PEP 8 (`snake_case`, nomenclatura semântica)
- Estruturação profissional: `main()`, `if __name__ == "__main__":`

## Como Executar

```bash
python Decisoes.py
```
