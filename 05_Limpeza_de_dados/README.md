# 📘 Exercício 05: Strings, Regex e Limpeza de Dados

## 🎯 Objetivo

Simular uma etapa crítica de ETL: **Data Cleaning**. Receber registros desestruturados ou "sujos" (provenientes de sistemas legados, planilhas manuais ou APIs heterogêneas) e aplicar regras de padronização, validação e formatação para garantir qualidade e consistência nos dados de saída.

## 📚 Conceitos Aplicados

- 🔹 Manipulação avançada de strings (`.strip()`, `.split()`, `.join()`, `.title()`, `.lower()`)
- 🔹 Expressões Regulares (Regex) com módulo `re` para extração e limpeza de padrões
- 🔹 Validação lógica de campos (email, CPF, telefone)
- 🔹 Normalização de dados para consistência de chaves (ex: emails em minúsculo)
- 🔹 Uso de `None` para representar dados inválidos (padrão SQL/NULL)
- 🔹 Composição de funções para orquestração de pipeline de limpeza

## 🚀 Como Executar

```bash
python limpeza-de-dados.py
```
