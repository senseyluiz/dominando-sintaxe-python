# 📘 Exercício 01: Variáveis, Tipos e Formatação de Dados

## Objetivo

Simular a etapa inicial de um pipeline de dados: **receber dados brutos como texto, converter para tipos adequados, tratar inconsistências e formatar para exibição ou carga posterior**, seguindo padrões de código limpo e reutilizável.

## Conceitos Aplicados

- Tipos primitivos: `str`, `int`, `float`, `bool`
- Conversão explícita e tratamento de entradas brutas
- Métodos de string: `.title()`, `.lower()`
- Formatação avançada com `f-strings`
- Estruturação profissional: `main()`, `if __name__ == "__main__":`

## Como Executar

```bash
python Variaveis-tipos.py
```
