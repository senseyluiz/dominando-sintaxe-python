# Exercício 03: Estruturas de Dados e Controle de Estoque

## Objetivo

Simular um sistema básico de gestão de inventário utilizando dicionários aninhados e fluxo interativo, aplicando validação robusta de entrada, tratamento de exceções e princípios de organização de código para garantir consistência e resiliência.

## Conceitos Aplicados

- Dicionários e acesso a valores aninhados (`estoque[produto]["quantidade"]`)
- Laços de repetição (`while True`) para sessões interativas contínuas
- Tratamento de exceções (`try/except`) para validação de tipos numéricos
- Normalização de strings (`.strip().lower()`) para consistência de chaves
- Princípios DRY e KISS na estruturação de funções independentes

## 🚀 Como Executar

```bash
python lista-dicionario.py
```
