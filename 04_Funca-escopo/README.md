# Exercício 04: Funções, Escopo e Documentação

## Objetivo

Refatorar lógica matemática e financeira em **funções modulares e reutilizáveis**, aplicando padrões de documentação (`docstrings`), anotações de tipo (`type hints`) e princípio DRY (Don't Repeat Yourself) para facilitar testes e manutenção futura.

## Conceitos Aplicados

- Definição e chamada de funções (`def`)
- Escopo de variáveis (Local vs. Global)
- Parâmetros com valores padrão (Default Arguments)
- Anotações de Tipo (`Type Hints`) para clareza de contrato
- Docstrings padrão para documentação automática
- Composição de funções (uma função chamando outra)

## 🚀 Como Executar

```bash
python funcao-escopo.py
```
